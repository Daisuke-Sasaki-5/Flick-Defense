# Flick-Defense  

# ゲーム概要  
* 上空から落ちてくる隕石を、フリックでミサイルを発射して迎撃するシンプルな防衛ゲーム

# 操作方法
* 画面の飛ばしたい方向にフリックしてミサイルを飛ばす

# 工夫した点
* 時間が経つたびに隕石のスポーン間隔を短くなるようにした

# 使用アセット
* ミサイルイラスト：イラストAC
* 隕石、ハートイラスト：ChatGPT
